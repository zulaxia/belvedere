Belvedere Automated File Manager

Belvedere is an automated file management application that performs actions on files based on user-defined criteria. For example, if a file in your downloads folder hasn't been opened in 4 weeks and it's larger than 10MB, you can tell Belvedere to automatically send it to the Recycle Bin. Belvedere is written by Adam Pash and distributed by Lifehacker under the GNU Public License. For details on how to use Belvedere, check out the Belvedere homepage.
