File Menu

The file menu has the following options available to the users:



Pause
Backup Settings...
Import Settings...
View Log...
Exit

Pause Menu Option

The pause menu option allows users to pause the application so that no folders or rules are processed until the user 'un-pauses' the application. A check mark next to the pause menu option in the File Menu, Tray Menu, and a RED tray icon denotes that Belvedere is in a paused state.

Top

Backup Settings... Menu Option

The Backup Settings menu option allows users to save the rules and application settings to a filename of their choosing.

Top

Import Settings... Menu Option

The Import Settings menu option allows users to overwrite the current application settings with that of ones saved in another file. This will not append or add rules/settings, but rather irreversibly overwrite existing settings.

Top

View Log... Menu Option
