Folders Tab

The Folders Tab is the main working area of the application. Here the users assign rules to the specific folders that they have selected.

The tab is divided into two main sections, the left half where the folders are selected, and the right, where the rules for the selected folder are displayed.

Folder List

This list shows the folders that have been added to Belvedere to monitor with the rules displayed on the right side.

The buttons to the left of the folder list allow the user to set the order in which folders, and their rules, are processed. Folder processing occurs from the top, down.

  • '/\' Button - Will move the selected folder up one position in the displayed list.
  • '\/' button - Will move the selected folder down one position in the displayed list.

The following buttons are available below the folder list:

  • '+' Button - To add a folder, click the '+' button at the bottom of the screen, and select the folder from the dialog box.
    • NOTE: Folders may also be added by dragging and dropping a file or folder to the above list box as well. The parent directory in each case will be the folder that is added.
  • '-' Button - To remove a folder, click the '-' button, and confirm your deletion of the folder. If you delete a folder with rules, the rules will be removed as well.

To display the rules assigned to a particular folder, left click to highlight the folder, and the assigned rules will be displayed in the right side of the application. The full path of the selected folder will also be displayed over the rules list.

Rule List

This list shows the rules for the folder selected in the left side of the screen and displayed immediately above. Here you can see the name of the rule, and if the rule is currently enabled or not. If a rule is listed as 'Yes' in the 'Enabled' column, the rules will be applied to all files within the specified folder. If the rule is listed as 'No' in the 'Enabled' column, this particular rule will be skipped over during processing.

The following buttons are available below the folder list:

  • '+' Button - To add a rule, click the '+' button at the bottom of the screen and complete the Create a rule... dialog.
  • '-' Button - To remove a rule, click the '-' button, and confirm your deletion of the rule.
  • Edit Rule Button - To edit an existing rule, highlight the rule above, and click the 'Edit Rule' button and complete the Edit Rule Dialog.
  • Enable/Disable Button - To enable or disable an existing rule, highlight the rule above, and click the 'Enable/Disable' button to change its status.

The buttons to the right of the rule list allow the user to set the order in which rules are processed. Rule processing occurs from the top, down.

  • '/\' Button - Will move the selected rule up one position in the displayed list.
  • '\/' button - Will move the selected rule down one position in the displayed list.

A context menu is available by right-clicking on the rule names in order to execute the following options:

  • Copy to - By highlighting 'Click to,' Belvedere will list the other folders to which you may copy the selected rule. An exact copy will be made with an incremented counter placed after the rule name to protect against duplicate names.
  • Move to - By highlighting 'Move to,' Belvedere will list the other folders to which you may move the selected rule. The chosen rule will be removed from the current folder and relocated to the newly chosen folder.
