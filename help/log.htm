Belvedere Log

The 'Belvedere Log' dialog is displayed by clicking the 'View Log...' option within the 'File' Menu. The user is then presented with the below dialog box.



The majority of this dialog box is used to display the contents of the log file, with the most recent log entry being at the very bottom of the list. The following buttons are also available to the user:

Refresh
Clear Log
Save Log...

Refresh Button

The refresh log button will refresh the current screen with the newest log entries. The screen will not autorefresh after each log entry, so it must be refreshed manually.

Top

Clear Log Button

The clear log button will irreversibly clear the application log file.

Top

Save Log... Button

The save log button allows users to save a copy of the current log file to another filename or directory of their choosing. This will not remove the current log, but rather make a copy of the current log and also keep it as the active log.
